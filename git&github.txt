Push the code to Git & Github Repostory
------------------------------------------------------------

1) Create local git repository for your project 
      git init  ---->Create an empty git repository(local Repository)

    
2) Connect your local Git(local repo) repo with Github(global repo)
 https://github.com/pravindesai2121/nopCommerceApp.git


Before doing commit first time we need to execute these below commands(its execute only one time)
        git config --global user.name "pravindesai2121"
        git config --global user.email "pravindesai2121@gmail.com"

3) Check the status of files(commited/not commited)
     git status

4) Add all the files to staging area  
    git add -A 

5) Commit the code into git repository(local repo)
    git commit -m "first commit" 

6) Push the code from git ---to---> github repository 
git push -u origin master
